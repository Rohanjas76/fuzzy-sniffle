# database/

Place one image per known person here.

## Naming convention
The filename (without extension) becomes the person's display name:

  elon_musk.jpg      →  "Elon Musk"
  marie_curie.png    →  "Marie Curie"
  john_doe.jpg       →  "John Doe"

## Tips
- One clear front-facing face per image
- Good lighting, eyes visible
- JPG, PNG, WEBP or BMP
- Any resolution works (face_recognition auto-scales)
